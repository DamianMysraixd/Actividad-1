Páginas web

página web es un documento digital accesible a través de Internet mediante un navegador web. Está diseñada utilizando
lenguajes de programación y marcado como HTML (HyperText Markup Language), CSS (Cascading Style Sheets) y JavaScript,
que en conjunto estructuran, estilizan y aportan funcionalidad a la página. Una página web puede contener texto, imágenes, videos, enlaces a otras páginas web, formularios interactivos, y otros elementos multimedia.
Las páginas web son componentes individuales de un sitio web, que es un conjunto de páginas relacionadas y
accesibles bajo un dominio común, como www.ejemplo.com. El contenido de una página web se almacena en un servidor web y se puede acceder a él a través de una URL (Uniform Resource Locator),
que es la dirección específica que un navegador utiliza para localizar y mostrar la página. Cuando un usuario solicita una página web introduciendo una URL en el navegador, éste envía una solicitud al servidor web,
que responde enviando el contenido de la página al navegador para que lo renderice y lo muestre al usuario. Las páginas web pueden ser estáticas o dinámicas. Las páginas estáticas muestran el mismo contenido a todos los usuarios,
mientras que las dinámicas pueden mostrar contenido personalizado basado en la interacción del usuario, como información de una base de datos, resultados de búsqueda, o contenido generado en tiempo real.

Hipervinculo

Un hipervínculo, también conocido como enlace o link, es un elemento en un documento digital que permite al usuario navegar a otra ubicación, ya sea dentro del mismo documento,
a otro documento en el mismo sitio web, o a una página web completamente diferente en Internet. Los hipervínculos son fundamentales para la estructura y la navegación en la web,
creando una red interconectada de información. Los hipervínculos pueden ser texto, imágenes, botones u otros elementos interactivos. En el caso del texto, se suelen destacar con un color diferente y, a menudo, con subrayado.
Al hacer clic en un hipervínculo, el navegador web redirige al usuario a la URL especificada en el enlace, cargando el contenido de destino. Desde un punto de vista técnico, un hipervínculo se define en HTML mediante la etiqueta ``, que contiene el atributo `href`
donde se especifica la URL de destino. Por ejemplo, Visitar youtube crea un enlace de texto que redirige al usuario a "https://www.ejemplo.com". Los hipervínculos mejoran la experiencia del usuario al permitir una navegación rápida y eficiente entre diferentes recursos,
facilitando el acceso a información relacionada y enriqueciendo el contenido con referencias externas. Además, son esenciales para el
funcionamiento de motores de búsqueda y para la creación de estructuras de sitios web bien organizadas y fáciles de explorar.
